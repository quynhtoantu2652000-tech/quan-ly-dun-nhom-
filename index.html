⚙️ Cấu hình Firebase

Để lưu bản vẽ tập trung và đồng bộ mọi thiết bị, cần kết nối Firebase (miễn phí). Làm theo 4 bước sau:

1
Tạo Firebase project miễn phí
Vào console.firebase.google.com → Tạo project mới → Đặt tên bất kỳ
2
Bật Firestore Database
Trong project → Build → Firestore Database → Create database → chọn test mode
3
Bật Authentication
Build → Authentication → Get started → bật Email/Password → thêm 1 user admin (email + password)
4
Lấy Firebase Config
Project Settings ⚙️ → Your apps → Web app () → copy firebaseConfig và dán vào ô dưới

Chế độ cục bộ: bản vẽ chỉ lưu trên thiết bị này, không đồng bộ các máy khác.
